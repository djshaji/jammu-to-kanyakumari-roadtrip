## Noise Analysis

### Type of Increased Noise
1. **Road Noise**
   - Tire noise from road surface
   - Engine noise transmission
   - Wind noise (possibly)

2. **Horn Noise**
   - Car's own horn audible inside cabin
   - Indicates sound insulation compromise

### Likely Root Causes

#### Dashboard-Related Issues
- [ ] **Sound dampening material** not reinstalled properly
- [ ] **Dashboard gaps** allowing sound transmission
- [ ] **Firewall insulation** disturbed during heater access
- [ ] **Door seals** affected by dashboard work

#### Missing/Displaced Components
- [ ] **Sound deadening foam** behind dashboard
- [ ] **Rubber grommets** for wire/cable passages
- [ ] **Insulation pads** around HVAC components
- [ ] **Sealing strips** between dashboard sections

#### Installation Issues
- [ ] **Dashboard not seated properly** against firewall
- [ ] **Clips and fasteners** missing or loose
- [ ] **Panel gaps** wider than original specifications
- [ ] **HVAC duct sealing** compromised
